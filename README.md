# HelloWorld

A very small beginner Python repo with two early practice scripts.

This repository is a learning exercise, not a full application. It shows basic variables, printing, user input, and f-string output.

## Files

```text
exercise.py   # Prints a name and age from variables
madlib.py     # Prompts for words and builds a short sentence
```

## How To Run

Run either script with Python:

```bash
python exercise.py
```

or:

```bash
python madlib.py
```

## What I Practiced

- Creating Python files
- Assigning variables
- Printing values
- Reading input from the user
- Formatting strings with f-strings
